## Premium($8)的真实成本

X 把 Premium 写成 "$8/月",但仔细看条件:

- $8 = 年付,折算月费;实际一次性付 $84/年
- 月付是 $11/月
- iOS 内购:网页 $8 -> iOS $11(苹果抽成)
- 年付 iOS:$112/年

实测组合:
- 最划算:年付网页版 $84,等于 $7/月
- 最常见:月付网页 $11(因为很多人不想锁一年)

**Premium+ 适合**:
- X 重度用户(每天发帖、看推、引战)
- 想用 Grok 4 的语音 / 视频生成功能
- 做内容创作希望 X 高分发

**Premium 适合**:
- 只想用 Grok 4 聊天 / 写作
- 偶尔 X(去广告就够)
- 不在乎语音 / 视频

## SuperGrok($300)谁会买

xAI 在 2025 年发的高端档,定位类似 ChatGPT Pro:

- 极高的 Grok 4 / Grok 4 Heavy 用量
- 多模型并行(同一个 prompt 同时跑几个模型再投票)
- 最优先 GPU 队列,峰期不排队
- 早期访问新模型(下一代 Grok 内测)

实测目标用户:专业研究者、AI 公司内部对比测试、xAI 重度粉。普通人不需要。

## Grok 跟 ChatGPT / Claude 怎么选

Grok 的核心差异化:

1. **实时性** —— 能直接读 X 上正在发生的事,新闻类问答比 Plus / Claude 快几小时甚至几天
2. **语气** —— 默认风格更"敢说话",政治议题、争议话题不像 ChatGPT 那么保守
3. **跟 X 深度集成** —— X 帖子分析、实时新闻分析是杀手级
4. **价格便宜** —— Premium $8 是同档最便宜的 AI 订阅(同时含 X 高级会员)

**适合订 Grok Premium 而不是 ChatGPT Plus 的人**:

- X 重度用户(每天看推 1+ 小时),Premium 等于 X 高级会员 + Grok 4 全套打包,$8 太划算
- 做新闻 / 实时事件分析(媒体人、研究者、二级市场分析师)
- 偏好不那么"政治正确"的回答风格

**不适合的**:

- 不用 X / Twitter 的人(Premium 一半价值是 X 会员功能)
- 写代码主力(Grok 4 编码能力比 Claude / GPT 弱一档)
- 需要稳定的图像 / 视频生成(Grok Imagine 进步快但稳定性不如 DALL·E / Veo)

## 国内订阅路径

最简单。前提是有海外卡 + 能登 X(国内被墙)。Stripe 通道,通过率类似 OpenAI(40-60%)。

### 路径 B:Apple ID 内购

绑美区 / 港区 Apple ID,X iOS App 内购。Premium $11、Premium+ $44(贵)。

### 路径 C:代充

国内主流。代充帮你登 X 账号 + 升级。价格便宜(因为 Grok 本身便宜),Premium 大约 80-100 RMB/月。

注意:

- 部分代充提供"X 账号 + 订阅打包",价格 150-200 RMB

### 中国大陆访问限制

X.com 在中国大陆完全不可访问,必须挂稳定的境外节点全程使用。Grok 网页版(grok.com)同样受限。

## Grok 数据训练 / 隐私

xAI 的隐私策略比 OpenAI / Anthropic 弱:

- X 上的公开帖子默认用于训练 Grok(2024-09 起)
- DM 和私密内容据称不训练,但用户没法选择关闭
- Premium / Premium+ 用户可以在 settings 关闭"使用我的数据训练 Grok"

如果你 X 账号涉及商业敏感、未发布内容、客户对话,建议:
- 关闭训练开关
- 不在 Grok 里粘敏感信息
- 用 xAI API 替代(API 默认不训练)
