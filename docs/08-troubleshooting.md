# 08 · 常见付款失败 / 风控 / 卡密问题排查

> 12 个真实场景,每条 2-3 步处方。从最常见的开始排,基本能覆盖 90% 的付款 / 激活问题。

## 场景 1:付款页转圈,最后报"Your card has been declined"

**最常见原因**:卡 BIN 段被风控、账单地址不匹配、IP 异常。

**处方**:

1. 先换稳定干净的美 / 日节点,挂稳 1-2 小时再付
2. 换一张卡(招行全币种 VISA 通过率最高)
3. 账单地址换真实美国地址 + 真实邮编(zip-state-city 三联要一致,不能填 90210 + Texas)
4. 还不行:走 Apple ID 礼品卡 / 代充

> 经验:同一张卡 24 小时内被拒 2 次,银行会自动锁住跨境消费,要打银行电话解除。别在被拒后立刻反复试。

## 场景 2:刚订成功,几小时 / 几天后被取消、收到 "Your subscription has been cancelled"

**最常见原因**:OpenAI / Anthropic 反查到付款卡是盗刷 / 共享 / 黑产。

**处方**:

1. 检查邮箱有没有正式取消邮件,看里面给的原因
2. 如果是"卡问题":钱可能已经退回,不算损失
3. 如果是"账号违规":申诉(help.openai.com -> Billing),但成功率不高
4. 之后换合法卡 / 代充重订

> 黑产卡的钱通常退回了,但账号可能进黑名单,要换新邮箱重注册。

## 场景 3:订阅成功了,但模型选择器里没看到 Plus / Pro 标识

**最常见原因**:OpenAI 状态同步延迟。

**处方**:

1. 等 5-30 分钟,刷新页面
2. 退出账号 + 重登
3. 清浏览器 cookie / 用无痕窗口登录
4. 还不行:开 https://chat.openai.com/api/auth/session 看 isPaid 字段是否是 true

如果 isPaid = true 但界面没解锁,纯粹是缓存问题,等就行。

## 场景 4:Apple ID 礼品卡显示"This card has already been redeemed"

**最常见原因**:卡密被人提前使用 / 假卡 / 二手卡。

**处方**:

1. 立刻截图卡密 + 兑换报错页,联系卖家
2. 找有担保的卖家(如淘宝七天无理由 / 闲鱼担保交易)能追回
3. 闲鱼 / 微信私下购买的,90% 找不回来,只能认栽
4. 之后买礼品卡只走"自动发货 + 平台担保"渠道

> 假卡市场 2025 后期变多,因为黑产用脚本批量爆破礼品卡库存,劣币驱逐良币。

## 场景 5:Apple ID 突然被锁,提示"Your Apple ID has been locked for security"

**最常见原因**:短时间多国 IP 登录、批量充值、多设备激活。

**处方**:

1. 在 https://iforgot.apple.com 走"重置密码"流程,通常 1-3 天能解
2. 解锁后立刻把双重验证手机号绑回正确的(很多人锁定就是因为手机号没绑)
3. 换稳定 IP,以后只在 1-2 个固定地区 IP 用
4. 如果是批量充值导致的"经济管制",联系苹果客服(中文也行)解释,大约 50% 概率解

## 场景 6:订阅显示成功扣款,但 Claude.ai / ChatGPT 提示"Subscription not found"

**最常见原因**:邮箱大小写不一致 / 账号绑定错。

**处方**:

1. 退出账号,用付款邮箱完全一致的小写形式重登
2. 检查 Settings -> Subscription,看是否真的关联到当前账号
3. 没有的话:联系官方客服(billing@anthropic.com / help.openai.com)说明扣款详情
4. 一般 1-3 天能恢复

## 场景 7:卡密激活后,几天就失效

**最常见原因**:黑产卡(共享号、盗刷号)被官方反查砍号。

**处方**:

1. 截图付款记录 + 激活记录,联系代充店要售后
2. 正规店:7 天 / 30 天保修期内会无偿换新
3. 跑路店 / 闲鱼店:基本要不回来,认损
4. 之后:挑代充看本仓库 [06-付款方式](./06-payment-methods.md#代充服务)

## 场景 8:Claude.ai 提示"Anthropic does not offer service in your country"

**最常见原因**:IP 落在 Anthropic 不支持的国家(中国大陆、俄罗斯、伊朗等)。

**处方**:

1. 挂稳定的 美 / 日 / 港 节点,确保 IP 干净
2. 测试节点是否真干净:访问 https://www.cloudflare.com/cdn-cgi/trace,看 loc 字段
3. 如果是云加速节点(IP 共享给大量用户),可能被 Anthropic 风控,换独享 IP / 换节点
4. 浏览器开无痕,清 Anthropic 域名所有 cookie 重登

## 场景 9:付款页提示"3D Secure verification failed"

**最常见原因**:卡的 3DS(动态密码 / 短信验证)没通过。

**处方**:

1. 检查银行 App 短信:跨境付款会触发短信 OTP,要在 5 分钟内点确认
2. 国内手机号有时收不到 OTP,需要联系银行客服开通跨境短信
3. 招行 / 中信:在 App 里手动开"跨境消费验证短信"
4. 还不行:换支持 Apple Pay 的卡,Apple Pay 走 Touch ID / Face ID 验证不依赖短信

## 场景 10:订阅了之后,IP 一切到国内立刻提示限制 / 被踢

**最常见原因**:OpenAI / Anthropic 检测到 IP 大幅漂移,触发"账号风控"。

**处方**:

1. 不要在国内 IP 直连,挂稳定境外节点全程使用
2. 节点不要频繁切换地区(美 -> 日 -> 港 -> 美 = 高风险)
3. iOS App / Android App 有时会泄露真实 IP(双层代理仍然漏),建议先用网页版
4. 重度用户用单独 ROUTING 把 AI 域名导出境外,其他流量走国内

## 场景 11:Cursor 订阅成功,但用 fast 请求时还是排队

**最常见原因**:Cursor 后端给 Pro 用户的 fast 配额是"软限制",高峰可能临时降级。

**处方**:

1. 登出账号 + 重登,刷新订阅状态
2. 检查 Settings -> Account 看是否真显示 Pro
3. 高峰期(美 PM 9-11 点 / 中 AM 9-11 点)排队不可避免,等几小时
4. 长期排队:升 Pro+ ($40),fast 配额翻 3 倍

## 场景 12:网页版 ChatGPT 已订阅但 iOS App 显示 Free

**最常见原因**:iOS App 跟网页版可能用了不同的账号 / 缓存。

**处方**:

1. iOS App 退出账号 + 重新登录
2. iOS Settings -> ChatGPT -> 清除应用缓存 / 重装
3. 检查邮箱大小写、是否登错了别的账号
4. 还不行:重启手机(iOS Apple Receipt 同步偶尔慢)

## 退款流程一览

| 场景                  | 推荐渠道                               | 成功率 |
| --------------------- | -------------------------------------- | ------ |
| 卡被风控扣了款没开通  | 银行客服争议交易                        | 70-90% |

| 代充店买的卡密 / 服务  | 联系代充店客服走售后                    | 看店家 |

## 防坑总结

1. **小额测试**:第一次订陌生服务,先小额($1-10)测试通过,再充满月费
2. **节点稳定**:整个付款流程同一个 IP,不要中途切节点
3. **不分享账号**:即使便宜,共享号短期会被砍,数据可能丢
4. **截图保留**:每一步操作截图,扯皮 / 退款 / 申诉都用得上
5. **挑代充看付款方式**:支持 USDT 的店通常更靠谱(付款方式越多越合规)
