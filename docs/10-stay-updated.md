## 新档位 / 调价 怎么第一时间发现

按信号源的"提前度":

1. **泄露(最早)**:OpenAI Pro 在正式发布前 3 周就有 reverse engineer 在 X 流出
2. **官方推文 / 博客(实时)**:订阅官号推送
3. **官方 changelog(同步)**:发布日同步更新
4. **大网站报道(+ 1-3 天)**:TechCrunch / The Verge / The Information
5. **中文翻译(+ 1-7 天)**:量子位、机器之心
6. **代充店调价(+ 1-14 天)**:这就是为什么订阅本指南 GitHub repo 也有用,价格表会更新

## 这份指南的更新机制

- 每月 1 日检查所有官方价格 / 档位,有变动则更新
- Issue / PR 收到后 1 周内处理
- 重大事件(新档位 / 大规模调价)2 天内出 [CHANGELOG](https://github.com/siuserxiaowei/awesome-ai-subscription/releases)

订阅本仓库 -> 点 GitHub 右上角 Watch -> Custom -> Releases,新版本会邮件通知。
