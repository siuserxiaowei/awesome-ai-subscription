## 性价比经验法则

1. **第一个 $20 永远是最值的**:$0 -> $20 体验提升 10 倍
2. **第二个 $20 价值递减但仍正**:$20 -> $40 体验提升 30-50%(主要在覆盖差异化场景)
3. **从 $20 跳到 $200 慎重**:$200 vs $20 的体验提升不到 5x,只有重度场景才值
4. **不要堆同档**:Plus + Pro 重叠太多没意义,要堆就堆"差异化大"的(Plus + Claude Pro)
5. **API 比订阅划算的临界点**:每天 < 30 分钟用量 -> API 便宜;每天 > 1 小时 -> 订阅便宜
6. **年付折扣不是越多越好**:AI 行业每 3-6 个月有大变化,锁年付可能错过更好的选择(Cursor 最近几次发新档都是月付能立刻切,年付的人吃亏)
