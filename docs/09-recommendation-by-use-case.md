# 09 · 按场景选订阅

> 5 类用户:程序员、学生、写作者 / 内容创作者、研究者 / 学术、普通用户。每类给具体的订阅组合 + 月预算 + 为什么这么选。

不写"看你需求自己选"这种废话。每个场景给你 3 档(够用 / 推荐 / 顶配),选一个买就行。

---

## 1. 程序员

### 够用版($20/月)

**Cursor Pro 单订**

为什么:
- Cursor 内置 Claude Sonnet / GPT-5 / Gemini 多家模型,IDE 内一站式
- 500 fast 请求 / 月,中等强度够用
- 不需要再单独订 ChatGPT / Claude(IDE 里面已经能用)

什么时候不够:重度 Agent / 长任务 / 大改代码,撞 fast 上限。

### 推荐版($40-60/月)

**Cursor Pro $20 + Claude Pro $20**(共 $40)

或

**Cursor Pro $20 + ChatGPT Plus $20 + Grok Premium 年付 $8**(共 $48)

为什么:
- Cursor:日常 IDE 编程主力
- Claude Pro:长代码 review / 项目级讨论 / 写文档(Projects 是杀手锏)
- 或加 ChatGPT Plus 用 Code Interpreter 跑数据分析、用 DALL·E 出图
- Grok Premium 8 块钱白嫖 X 实时新闻和 Grok 4 实时搜索

### 顶配版($120-300/月)

**Cursor Pro $20 + Claude Max $100**(共 $120)

或

**Cursor Pro+ $40 + Claude Max $200**(共 $240)

为什么:
- Cursor 做日常 IDE 工作
- Claude Max 给 Claude Code 命令行开大额度,跑长任务 Agent 不卡
- 重度 Agent 开发者(每天 Claude Code 跑 100+ 命令)选 $200 档

## 2. 学生

### 够用版($0-10/月)

**Free 全家桶 + Grok Premium 年付 $8**

为什么:
- ChatGPT Free / Claude Free / Gemini Free 三家叠用,一家撞额度切下一家
- Grok Premium $8 提供完整 Grok 4 + X 高级会员(社交媒体 + AI 一体)

学生场景 90% 是查资料 / 写作业 / 学新东西,Free 档基本够。

### 推荐版($20/月)

**Claude Pro 单订**(写作 / 研究主导)

或

**ChatGPT Plus 单订**(理工 / 编程主导)

为什么:
- 学生场景多是长文档(论文、教材)+ 写作业 -> Claude Pro 的 Projects + 200k 上下文是杀手锏
- 理工科学生写代码 / 跑数据 -> Plus 的 Code Interpreter 更顺手
- 不要订两个,$20 已经够覆盖大部分需求

### 顶配版($40/月)

**ChatGPT Plus + Claude Pro 双订**

为什么:
- 一个写作论文(Claude),一个查资料 / 编程 / 一般问答(ChatGPT)
- $40 是学生预算上限,再上去 ROI 就低了
- 不建议学生订 Pro $200 / Max $100,撞上限的概率极低

学生折扣:OpenAI 在 2024 年发过"学生 Plus 半价"试点,2026-05 仍偶尔重启,关注 https://openai.com/students。

## 三句话总结

1. 不知道选什么 -> **ChatGPT Plus $20** 最不会出错
2. 程序员 -> **Cursor Pro $20** 必备,有钱再加 Claude Pro
3. 长文 / 学术 / 内容创作 -> **Claude Pro $20**(配 Gemini AI Pro 更香)

不要堆 5 个订阅然后大半月不用。先订 1 个,1 个月内用满,撞瓶颈再加下一个。
