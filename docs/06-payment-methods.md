## 海外 / 国内信用卡

### 海外本土卡(Chase Sapphire / Amex Gold / 香港汇丰运通卡 等)

最稳。前提是你或亲友长居海外、能办本土卡。

- **被拒概率**:< 5%,基本无风控
- **额外成本**:0(美区订阅用美元卡、欧元区卡用 EUR 计价)
- **要点**:账单地址跟卡完全一致;别用同一张卡 1 个月内开 5+ 个 AI 订阅(银行可能查盗刷)

如果是这条路,本节其他可以不看。

### 国内信用卡的真实通过率

国内银行发的"全币种" / "VISA / MC" 卡能不能用?要分情况。实测 2026-05 数据:

| 卡片                            | OpenAI 通过率 | Stripe 通用通过率 | 备注                       |
| ------------------------------- | ------------- | ----------------- | -------------------------- |
| 招商银行 全币种 VISA / MC       | 60-70%        | 70-80%            | 成功率最高的国内卡         |
| 中信 i 白金 / Lily / 万事达     | 50-60%        | 60-70%            | 几年来一直是热门           |
| 华夏精英卡 VISA                 | 40-50%        | 50-60%            | 看 BIN 段                   |
| 浦发 AE 白(运通)              | 30-50%        | 50-65%            | 最近开始好转                |
| 工行 VISA / MC                  | 20-30%        | 30-40%            | 因为 BIN 大量被风控         |
| 建行 / 农行 VISA / MC           | 10-25%        | 20-40%            | 风控较严                    |
| 银联单标卡(纯银联,非 VISA/MC) | < 5%          | 0%                | 海外商户基本不收            |
| 支付宝 / 微信信用卡             | 0%            | 0%                | 不出境支付                   |

**为什么国内卡会被拒**?

1. 国内卡的 BIN 段被发卡机构标记为"高风险出境",OpenAI / Stripe 对这些 BIN 直接拒
2. 账单地址 / 邮编不匹配 —— 中国地址在 OpenAI 的 zip 数据库不存在,填美国地址又跟卡注册不一致
3. IP 异常 —— 卡是国内卡但 IP 是美国(代理),触发 "card not present + suspicious IP" 双重风控
4. 银行端的境外消费保护 —— 部分银行默认关闭"未授权 / 无 OTP 跨境扣款"

**提高通过率的具体步骤**:

1. **节点先稳定**:挂一个干净的美 / 日节点,挂稳几小时再去付(IP 短期浮动会触发风控)
2. **账单地址用真实美国地址**:可以用真实公开地址(图书馆、公司),邮编 zip 必须跟 city / state 数据库一致
3. **银行短信确认**:很多卡跨境扣款会发短信验证,要在短信内点确认链接(国内手机号一般能收)
4. **小额测试**:先订一个 $1-5 的便宜服务(比如某个域名)成功一次,再回头订 ChatGPT,银行白名单生效后通过率涨一档
5. **不连续测试**:被拒后别立刻再试,银行会在 24 小时内对同一商户拒卡 2-3 次直接锁卡

## Apple Pay / Apple ID 礼品卡

最适合:不想搞海外信用卡 + 主要用 iOS / iPadOS。

**流程**:

1. 注册新美区 Apple ID(不能用国区 Apple ID 换地区,要新建)
2. 淘宝 / 闲鱼买美区 App Store 礼品卡(实卡截图卡密)
3. App Store -> 礼品卡 -> 兑换 -> 充入余额
4. App 内购订阅,扣余额

**优点**:

- 通过率极高(< 10% 被拒,大多是 Apple ID 信用问题)
- 不需要海外信用卡 / 海外手机号
- 苹果客服中文,出问题能沟通

**缺点**:

- 苹果抽 30%,Plus 从 $20 变 $25(贵 $5/月,一年贵 $60)
- 美区 Apple ID 自己要小心维护(短时间 IP 漂移、批量充值会被锁)
- 礼品卡偶尔遇到"已被使用 / 已冻结"的卡(假卡 / 二手卡),要找有信誉的卖家

**避坑**:

- 礼品卡尽量买 Apple 官方 / 正规零售商售出的(美国 Costco、Best Buy、Apple.com 直发)
- 淘宝礼品卡确认能 100% 兑换再支付(电子卡密走第三方担保)
- 一次充 $50-100,够 2-4 个月订阅,不要一次充 $500 引人注意

## Google Play 礼品卡(Android)

跟 Apple ID 礼品卡同理,但用谷歌账号:

- 通过率比 Apple 略低(10-20% 被拒,谷歌账号风控比苹果严)
- 抽成同样 30%
- 谷歌礼品卡市场不如 Apple 礼品卡稳定,假卡更多

## 虚拟卡(Wise / Revolut / Wildcard / Cardvcc 等)

虚拟卡是开一张"美 / 欧 / 港"的卡号,绑定到自己的 Wise / Revolut / Wildcard 账户,可以用来付海外订阅。

### Wise

- 注册需海外手机号 + KYC(护照 / 海外身份证 / 海外地址证明)
- 国内用户拿到 Wise 比较麻烦,过去几年门槛涨了一档
- 一旦开通,稳得不行,通过率 95%+
- 汇率好(中间价 + 0.4-0.7%)

### Revolut

- 类似 Wise,以英国 / 欧盟为主
- 国内用户 KYC 难度大
- 稳定性高

### Wildcard / Cardvcc / OneKey 等"虚拟卡平台"

中文圈最常用,因为 KYC 门槛低:

- 注册往往只要邮箱 + 护照(部分免 KYC)
- 平台开"美区虚拟卡",充 USDT 即可
- 通过率:60-80%(比国内卡高,但低于 Wise)
- 风险:平台跑路 / 卡突然失效 / 商户拒收某些 BIN
- 月费:开卡费 $1-10,平台抽 1-3% 充值费

**热门虚拟卡平台 2026-05 现状**:

| 平台         | 主要特点                            | 风险点                              |
| ------------ | ----------------------------------- | ----------------------------------- |
| Wildcard     | 中文圈用户最多,充 USDT             | 卡 BIN 段经常被新拒,要换卡         |
| OneKey Card  | OneKey 钱包配套,稳定性较好          | 需要装 OneKey 钱包                   |
| Nobepay      | 简单粗暴,充 USDT 开卡               | 部分商户已拒该 BIN 段                |
| dcard / vcc.com 系列 | 早期玩家,部分仍稳        | 散户多,合规性差                     |

**经验法则**:虚拟卡是"中等成本中等通过率"的中间方案。如果你已经熟练用代充或有 Wise,虚拟卡价值就不大。

## USDT / 加密货币付款

部分代充店(包括 [payforgpt.com](https://payforgpt.com))接受 USDT 付款。

**流程**:

1. 自己有 USDT(交易所 / 钱包)
2. 提到代充店给的 TRC20 / ERC20 / Polygon 地址
3. 链上确认后(TRC20 ~30 秒,ERC20 几分钟),代充店开始处理订阅

**优点**:

- 不涉及境内银行,通道完全外置,零风控
- 不可逆:下完单代充店没法跑路(大店家不会因为 $20 跑路)
- 转账成本极低(TRC20 大约 $1 / 笔)

**缺点**:

- 自己要有加密货币 + 钱包基础知识
- 链上汇率有 0-1% 波动
- 一旦发错地址不可追回

**适合**:已经在用加密货币的用户。完全不接触加密货币的纯小白门槛较高。
