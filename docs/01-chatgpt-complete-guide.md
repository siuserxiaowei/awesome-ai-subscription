> 适用范围:OpenAI 旗下 ChatGPT 网页版 / iOS / Android / 桌面客户端的所有付费档位。本篇不覆盖 OpenAI Platform / API(那是按 token 计费,跟订阅无关)。
> 数据时间:2026-05。OpenAI 大约每 6-9 个月调一次档位结构,过期请提 issue。

OpenAI 当前一共 5 个档位,从免费到企业级:

## Free vs Plus —— $20 买到了什么

很多人纠结要不要订 Plus,以下是 Free 跟 Plus 在 2026-05 的实际差异:

| 维度           | Free                     | Plus ($20)                                          |
| -------------- | ------------------------ | --------------------------------------------------- |
| 默认模型       | GPT-5 mini               | GPT-5(智能版)+ o4 推理模型                         |
| 用量限制       | GPT-5 每 5 小时 ~10 条   | GPT-5 每 3 小时 80 条;o-系列每周 ~50 条             |
| 高级语音       | 限量 / 短时              | 几乎不限                                            |
| DALL·E 图像     | 每天 2-3 张限量          | 每 3 小时 ~50 张                                    |
| Code Interpreter / 数据分析 | 限量          | 完整可用                                            |
| 自定义 GPTs    | 只能用别人的             | 可以创建 + 用别人的                                 |
| 文件上传       | 受限                     | 单次最多 10 个文件,单文件 512 MB                    |
| Deep Research  | 月 5 次轻量              | 月 25 次完整 + 25 次轻量                            |
| Sora 视频      | 不能用                   | 月 50 个 720p 视频(普通版)                         |

**$20 主要买的是用量解放和模型解放**。如果一周里你跟 ChatGPT 说话超过 1 小时,几乎一定会撞 Free 的限额,这时候 Plus 是必选。

## Plus vs Pro —— $200 那 10 倍的钱花在哪

Pro 是 2025 年 12 月发布的高端档位,一个月 $200,跟 Plus 比有 5 个核心区别:

1. **GPT-5 Pro 模式独占**:GPT-5 Pro 是后台用 think-and-reflect 多次推理后再输出,适合复杂数学、科研推理。Plus 用户用不了。
2. **o3-pro / 新版 o-系列推理模型不限量**:Plus 一周给你 50 条 o4 / o3,Pro 直接放开。
3. **Operator(浏览器代理)更高额度**:能让 ChatGPT 自己开浏览器买票、订外卖、填表单。Plus 限量,Pro 几乎敞开。
4. **Deep Research 几乎不限**:Plus 月 25 次,Pro 月 250+ 次,实测一天能跑十几个 5 分钟的深度研究任务。
5. **Sora 优先生成 + 无水印**:Pro 能跑 1080p、20 秒视频,Plus 只到 720p / 10 秒。

**什么人值得 Pro?**

- 一天跑 3+ 次 Deep Research(单次 5-15 分钟,Plus 一个月就用完了)
- 重度做数学 / 科研 / 编译器分析,要 GPT-5 Pro 模式
- 业务流要用 Operator 大量做浏览器自动化
- 在做 AI 视频内容,Sora 是核心生产力

**什么人不值得?**

- 日常写作、写代码、跟 AI 聊天 —— Plus 完全够,Pro 多花的 $180 全是边际浪费
- 只是觉得"贵的肯定好" —— 实测 Plus 用户里 90% 升级到 Pro 一个月内取消

## Team 档位 —— 谁用得上

Team 是面向 2-150 人小团队的方案,$25/人/月(年付)。3 个亮点:

1. **数据默认不用于训练** —— Plus 默认也不训练但开关在用户手里;Team 是组织级强制
2. **共享 GPT 库** —— 团队建的自定义 GPT 自动同步给所有成员
3. **管理员后台** —— 邀请成员、看用量、统一付费

注意:Team 最低人数是 2 人,一个人订不了。如果你是单人用户,直接 Plus 就行,Team 多花的钱买的是"管理"不是"模型"。

## Enterprise —— 一般人不用看

100+ 人公司,SSO、SCIM、审计日志、SLA、定制额度,没有公开报价(实测 100 人起报、$60-80/人/月浮动)。流程长:申请 -> 销售对接 -> 签 MSA -> 部署 -> 上线,平均 4-6 周。一般用户接触不到。

## 怎么订阅(按地理位置和支付能力分类)

### 路径 A:在海外 / 有海外信用卡

最直接:登录 ChatGPT -> 点 Upgrade -> 填卡号(Visa / Master / AmEx 都行)-> 付完即开通。

注意:
- 账单地址要跟卡的发卡地一致(US 卡填 US 地址、邮编要对得上 zip 数据库)
- IP 不能在 OpenAI 不支持的地区(中国大陆、俄罗斯等)。境内 IP + 海外卡也会被拒。

### 路径 B:国内 + 海外信用卡(中信白金、招行 VISA 全币种、华夏 Mastercard 等)

实测通过率:大约 40-60%。常见拒因:
- 卡的 BIN 落在国内段,被 OpenAI 风控直接拒
- 账单邮编填错(招行/中信信用卡很多人不知道账单邮编是开户地址邮编,不是身份证地址)
- IP 是境内动态 IP,触发风控

成功的卡:招行全币种 VISA、中信 i 白金、华夏精英卡(部分)、浦发 AE 白(少量成功)。
失败率高的卡:工行万事达、建行龙卡、农行卡、各家国内段的银联卡(银联卡基本必拒)。

成功率提升技巧:
1. 全程挂稳定干净的美国 / 日本节点(避免共享 IP 池)
2. 账单地址用真实美国地址 + 真实美国 zip(可用 fakeaddress.info 但注意现在 OpenAI 会校验 zip-state-city 三联)
3. 卡里至少有 $25(预扣 $1 + $20 月费 + 缓冲)

### 路径 C:Apple ID 内购

绑美区 / 港区 / 日区 Apple ID,App Store 充礼品卡,App 内订阅。

优点:不需要海外信用卡。
缺点:贵 25%(苹果抽成,Plus 变 $25/月);Apple ID 自己要新建美区(国区不能切美区订阅 ChatGPT)。

## 国内常见问题

### 问题 1:付款页转圈,最后报"Your card has been declined"

90% 是卡 BIN 段被风控。换卡(优先国际段全币种卡)。或换路径(Apple ID / 代充)。

### 问题 2:订阅成功了,但模型选择器里没 GPT-5 / Plus 标识

清浏览器 cookie + 重启 App。OpenAI 的状态同步偶尔延迟 5-30 分钟。还不行就退出账号重登。

### 问题 3:订阅了几天突然被取消、提示"违反 ToS"

最常见原因:
1. **账号是别人共享的** —— OpenAI 会扫账号 IP 漂移,一个号一周内出现 5+ 国 IP 直接砍订阅
2. **付款卡是盗刷卡** —— 跨境黑产卡,OpenAI 跟 Stripe 联动反查,一锁连账号一起
3. **账号被举报滥用** —— 大量调 API 风格请求、自动化、违禁内容

预防:一号一卡一稳定 IP,不要分享。

### 问题 4:被风控砍号 / 卡片拒付,有没有自助申诉

有,但成功率低。流程:
1. https://help.openai.com 提交工单,选 "Billing"
2. 用英文说明问题,附上交易日期 + 卡尾号(不要给完整卡号)
3. 平均 2-7 天回复

实测:卡风控被拒的,90% 申诉无效;账号砍号的,40% 能恢复。

## 退款 / 取消

- **取消**:Settings -> Subscription -> Manage -> Cancel。当月剩余天数继续可用,下月不再扣费。
- **退款**:Plus 没有官方退款政策(订阅类),但联系客服说明"刚订就要退",大约 30% 概率全额退,前 7 天内成功率最高。
- **Apple 内购**:走 reportaproblem.apple.com 申请,iOS 退款比网页好操作得多。
