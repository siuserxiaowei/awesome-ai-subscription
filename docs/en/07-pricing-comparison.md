## Value rating

Ranked by "value per $1" as of 2026-05:

1. **Grok Premium annual $8/month**: includes both X Premium membership + full Grok 4. Effectively $4 X membership + $4 AI. Pricing is borderline unbelievable.
2. **GitHub Copilot Pro $10**: saves a developer 4 hours per 2 hours of work. $10/month is essentially free.
3. **Gemini AI Pro $19.99**: 2TB cloud + Workspace + NotebookLM bundled. Each piece alone would cost $10+.
4. **ChatGPT Plus $20** / **Claude Pro $20**: most complete feature sets in their tier; well-differentiated.
5. **Cursor Pro $20**: core productivity for developers. $20 is a no-brainer.
6. **Notion AI $10**: only worth it for heavy Notion users.
7. **Perplexity Pro $20**: search is unique, but overlaps ~60% with ChatGPT search.
8. **Claude Max $100**: required for specific scenarios. Not for general users.
9. **ChatGPT Pro $200**: Deep Research + Sora + Pro reasoning trio. Only worth it if you need all three.
10. **Gemini AI Ultra $249.99**: Veo 3 is the core. Worth it for video creators, not for others.

## Recommended bundles by budget

### Budget $10/month

**Only choice**: Grok Premium annual at $8 (monthly equivalent).

The remaining budget covers occasional OpenAI API usage as a backup.

### Budget $20/month

Pick one of three:

- **ChatGPT Plus** — most well-rounded; daily chat / writing / light coding all covered
- **Claude Pro** — long documents / writing / Projects killer feature
- **Cursor Pro** — only choice for developers

Non-developers should pick ChatGPT Plus or Claude Pro. Subscribe to one for a month and see which fits.

### Budget $30-40/month

Add one complement:

- **Plus + Grok Premium $8** = $28/month. General AI + X real-time news.
- **Claude Pro + Grok Premium** = $28/month. Long-form writing + real-time news.
- **Cursor Pro + Claude Pro** = $40/month. IDE coding + long documents (standard combo for many developers).

### Budget $50-60/month

The trio:

- **Plus + Claude Pro + Grok Premium** = $48/month. Best general coverage (currently the most popular bundle).
- **Cursor Pro + Claude Pro + Plus** = $60/month. Developer deluxe.
- **Plus + Gemini AI Pro + Grok Premium** = ~$48/month. General AI + Workspace integration.

### Budget $100/month

Now you're in "all scenarios covered":

- **Plus + Claude Pro + Cursor Pro + Gemini AI Pro + Grok Premium** = $88/month. All platforms unlocked.
- **Plus + Claude Max $100 + Grok Premium** = $128/month (slight overage). Heavy Claude users.
- **Cursor Pro + Claude Max $100** = $120/month (slight overage). Developer with deep agent use.

### Budget $200+/month

Actively choosing Pro / Max high-end tiers:

- **Claude Max $200 + Plus $20 + Cursor Pro $20** = $240/month. Claude-centric.

### Budget $300+/month

Only "professional / heavy commercial" use cases reach this tier:

- **Pro $200 + Max $100 + Cursor Pro+ $40 + Gemini Ultra $250 (for video) + Grok Premium+ $40** = $630/month.

At this budget you should reconsider: **API + your own agent stack** instead of stacking subscriptions. $600/month buys far more API usage than subscriptions.

## Value-for-money rules of thumb

1. **The first $20 is always the best value**: $0 → $20 = 10x experience improvement
2. **The second $20 has diminishing returns but still positive**: $20 → $40 = 30-50% improvement (mostly differentiated coverage)
3. **Jumping from $20 to $200 — be careful**: $200 vs $20 is less than 5x experience gain. Only worth it for heavy use.
4. **Don't stack the same tier**: Plus + Pro overlap too much to make sense. Stack differentiated products (Plus + Claude Pro).
5. **API vs subscription threshold**: < 30 min/day → API is cheaper; > 1 hour/day → subscription wins.
6. **Annual discount isn't always better**: AI moves every 3-6 months. Locking annual may mean missing better options. Cursor's recent tier launches have made monthly subscribers winners — annual users got stuck.
