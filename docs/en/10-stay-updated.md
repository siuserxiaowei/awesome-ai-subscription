## How to spot new tiers / price changes first

By signal "earliness":

1. **Leaks (earliest)**: OpenAI Pro had reverse-engineered leaks on X 3 weeks before official announcement
2. **Official tweets / blog (real-time)**: subscribe to official accounts for push notifications
3. **Official changelog (sync)**: updated on launch day
4. **Major media coverage (+ 1-3 days)**: TechCrunch / The Verge / The Information
5. **Chinese translation (+ 1-7 days)**: QbitAI, Synced
6. **Subscription proxy price adjustments (+ 1-14 days)**: this is also why subscribing to this GitHub repo helps — pricing tables update here

## Update mechanism for this guide

- Check all official prices / tiers on the 1st of each month, update if changed
- Issues / PRs handled within 1 week
- Major events (new tier / large-scale price changes) get a [CHANGELOG](https://github.com/siuserxiaowei/awesome-ai-subscription/releases) entry within 2 days

Subscribe to this repo → click GitHub top-right Watch → Custom → Releases. New versions notify by email.
