# 09 · Pick a Subscription by Use Case

English | [中文](../09-recommendation-by-use-case.md)

---

> Five user types: developers, students, writers / content creators, researchers / academics, general users. Each gets a concrete subscription combo + monthly budget + reasoning.

No "depends on your needs, you decide" filler. Each scenario gives you 3 tiers (enough / recommended / max). Pick one and subscribe.

---

## 1. Developers

### Enough ($20/month)

**Cursor Pro alone**

Why:
- Cursor has Claude Sonnet / GPT-5 / Gemini multi-model built in, one-stop in IDE
- 500 Fast requests / month covers medium-intensity work
- No need to separately subscribe to ChatGPT / Claude (already accessible inside the IDE)

When it isn't enough: heavy agent / long tasks / large refactors hit the Fast cap.

### Recommended ($40-60/month)

**Cursor Pro $20 + Claude Pro $20** (total $40)

Or

**Cursor Pro $20 + ChatGPT Plus $20 + Grok Premium annual $8** (total $48)

Why:
- Cursor: daily IDE coding workhorse
- Claude Pro: long code review / project-level discussion / writing docs (Projects is killer)
- Or add ChatGPT Plus for Code Interpreter for data analysis, DALL·E for diagrams
- Grok Premium for $8 = X real-time news + Grok 4 live search

### Max ($120-300/month)

**Cursor Pro $20 + Claude Max $100** (total $120)

Or

**Cursor Pro+ $40 + Claude Max $200** (total $240)

Why:
- Cursor for daily IDE work
- Claude Max gives Claude Code CLI a high quota — long agent tasks don't stall
- Heavy agent developers (100+ Claude Code commands daily) should choose the $200 tier

## 2. Students

### Enough ($0-10/month)

**Free everywhere + Grok Premium annual $8**

Why:
- ChatGPT Free / Claude Free / Gemini Free — stack all three, switch when one hits the limit
- Grok Premium $8 gives full Grok 4 + X Premium membership (social media + AI in one)

90% of student use cases are research / homework / learning. Free tiers cover most of it.

### Recommended ($20/month)

**Claude Pro alone** (writing / research focused)

Or

**ChatGPT Plus alone** (STEM / coding focused)

Why:
- Student work is mostly long documents (papers, textbooks) + homework → Claude Pro's Projects + 200k context is the killer feature
- STEM students who code / process data → Plus's Code Interpreter is smoother
- Don't subscribe to two — $20 already covers most needs

### Max ($40/month)

**ChatGPT Plus + Claude Pro dual subscription**

Why:
- One for paper writing (Claude), one for research / coding / general Q&A (ChatGPT)
- $40 is the student budget ceiling. Above that, ROI drops sharply.
- Don't subscribe to Pro $200 / Max $100 as a student — hitting the cap is extremely unlikely

Student discount: OpenAI ran a "student Plus half-price" pilot in 2024. As of 2026-05 it occasionally restarts. Watch https://openai.com/students.

## 3. Writers / content creators

### Enough ($20/month)

**Claude Pro alone**

Why:
- Claude is the consensus pick for "most consistent style, least filler" on long-form
- Projects lets you put "your writing style + past work" into context. Output reads more like you.
- Artifacts gives a fluid edit-as-you-write experience, smoother than ChatGPT

### Recommended ($40-50/month)

**Claude Pro + ChatGPT Plus** (total $40)

Or

**Claude Pro + Gemini AI Pro** (total $40)

Why:
- Claude for long-form + ChatGPT DALL·E for cover images / Advanced Voice for brainstorming
- Or Claude for long-form + Gemini editing in Google Docs (Workspace integration) + NotebookLM for research

### Max ($120-280/month)

**Claude Max $100 + ChatGPT Plus $20** (total $120)

Or add **Gemini AI Ultra $250** (for video content)

Why:
- High-volume writers (5000+ words/day) hit Pro's quota. Max $100 gives 5x.
- Video creators (YouTube / TikTok) — Veo 3 is core productivity, worth Ultra

## 4. Researchers / academics / knowledge workers

### Enough ($20/month)

**Gemini AI Pro alone**

Why:
- NotebookLM is a killer tool for researchers: drop in 50 papers, chat directly with citations
- 2TB cloud storage for the literature
- Workspace integration for writing in Docs

### Recommended ($40-60/month)

**Gemini AI Pro + Claude Pro** (total $40)

Or

**Gemini AI Pro + ChatGPT Plus + Grok Premium** (total $48)

Why:
- Gemini NotebookLM for literature research
- Claude Pro for long documents / lit reviews (200k context beats ChatGPT)
- ChatGPT Code Interpreter for processing experimental data / generating paper figures
- Grok for real-time news (research tracking)

### Max ($200-400/month)

**ChatGPT Pro $200 + Gemini AI Pro $20 + Claude Pro $20** (total $240)

Why:
- ChatGPT Pro's Deep Research at 250+ runs/month is a force multiplier for academic literature reviews
- Gemini for the literature library, Claude for specific long documents
- This combo is standard for research-intensive / academic-writing-intensive scenarios

---

## 5. General users (daily Q&A / life assistant / translation / learning)

### Enough ($0/month)

**ChatGPT Free + Claude Free + Gemini Free, stacked**

Why:
- Free tier is enough for casual daily Q&A
- Switch when one hits its limit
- Don't spend money

### Recommended ($20/month)

**ChatGPT Plus alone**

Why:
- Most well-rounded, most general-purpose, most mature ecosystem
- Advanced Voice + DALL·E + Code Interpreter + Custom GPTs all useful (you won't hit all daily, but you'll need each at some point)
- Chinese experience is comparable to Claude — but Plus covers more daily use cases

### Max ($30-40/month)

**ChatGPT Plus + Grok Premium** (total $28)

Or

**ChatGPT Plus + Gemini AI Pro** (total $40)

Why:
- Grok for real-time news + X integration, complements Plus
- Or Gemini for 2TB cloud + possibly Workspace use — more practical for daily life than Grok
- Don't spend more. $40 is the ROI ceiling for general users.

---

## Three-line summary

1. Don't know what to pick → **ChatGPT Plus $20** is the safest bet
2. Developer → **Cursor Pro $20** mandatory; add Claude Pro if budget allows
3. Long-form / academic / content creation → **Claude Pro $20** (paired with Gemini AI Pro is even better)

Don't stack 5 subscriptions and use 2 of them. Subscribe to one, fully utilize it for a month, only add another when you hit a real bottleneck.
