## 贡献

发现错误、价格变动、新档位、付款新姿势,都欢迎 PR。看一眼 [CONTRIBUTING.md](./CONTRIBUTING.md) 30 秒就懂规则。

最受欢迎的贡献类型:
- 价格更新(各家每年至少调价 1-2 次)
- 国家/地区限制变化(例:Gemini 在某些地区开放/关闭)
- 新付款渠道实测反馈
- 风控触发的新场景

不接受的:广告软文、未经验证的二手信息、纯 AI 生成的洗稿内容。
